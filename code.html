                        // Get the String from the page
                        // Controller Function
                        function getString(){
                        
                            document.getElementById("alert").classList.add("invisible");
                        
                            let userString = document.getElementById("userString").value;
                        
                            // check for palindrome
                            let returnObj = isPalindrome(userString);
                        
                            // display message to the screen
                            display(returnObj);
                        }
                        
                        // Reverse the String
                        // Logic Function
                        function isPalindrome(userString){
                        
                            userString = userString.toLowerCase();
                        
                            // remove spaces and special characters
                            let regex = /[^a-z0-9]/gi;
                            userString = userString.replace(regex, "")
                        
                            let revString = [];
                        
                            let returnObj = {
                                msg: "",
                                reversed: ""
                            };
                        
                            for (let index = userString.length - 1; index >= 0; index--) {
                        
                                revString += userString[index];
                            }
                        
                            returnObj.reversed = revString;
                        
                            if (revString == userString){
                                returnObj.msg = "Well done! You entered a Palindrome!";
                            } else {
                                returnObj.msg = "Sorry! You did not enter a Palindrome!";
                            }
                        
                            
                        
                            return returnObj
                        }
                        
                        // Display the Reversed String
                        // View Function
                        function display(returnObj){
                        
                            document.getElementById("alertHeading").innerHTML = returnObj.msg;
                            document.getElementById("msg").innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;
                        
                            // Show the alert box
                            document.getElementById("alert").classList.remove("invisible");
                        
                        }
                        
                        function clear(){
                        
                            document.getElementById("userString").value = "";
                            document.getElementById("alert").classList.add("invisible");
                        }

The Code is structured in four functions.

getString()

Get String is the controller function. It is responsible for making sure the alert is set to invisible, getting the user input from the page, assigning it to the userString variable, passing the variable to the isPalindrome logic function and assigning the result to a new return Object, and lastly calling the display function to show the result to the user.

isPalindrome(userString)

Revers String is the logic function. It takes one parameter which is the string entered by the user. The first step in this function is to clean up the user input and remove any special characters. This is accomplished by creating a regex variable that contains the acceptable characters and using the replace method against the user string to replace all special characters with an empty string. The function then creates and empty array to store the reversed version of the input string and an object to hold the message string and the reversed string. A for loop is then used to start at the last position of the user string and work backwards adding each letter to the reversed string array. Once the loop concludes the reversed string is compared to the user string. Depending on if they match or not will determine what message is added to the return Object. The return object is then returned.

display(returnObj)

Display String is the View function. It takes one parameter which is the return object created by the isPalindrome(userString) function. The function then adds the two class attributes, which are the message and the userString to the alert box and removes the invisible class from the class list making the alert box visible.

clear()

The clear function takes not parameters. It is responsible for clearing the input field and making the alert box invisible again.
